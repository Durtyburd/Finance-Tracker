Finance Tracker (React + Vite) | https://durtyburd.github.io/Finance-Tracker/ (live site)

A simple finance tracking web application built with React and Vite that allows users to view, process, and analyze financial transactions. This project focuses on demonstrating practical JavaScript logic, React component structure, and state management.

Features

- Upload financial data (JSON or future PDF support)

- Display transactions in a clean UI

- Automatically calculate totals

- Format currency values to two decimal places

- Dynamic rendering using React components

Tech Stack

- React

- Vite

- JavaScript (ES6+)

- CSS (Still deciding on which framework to use)

- Node / npm

Getting Started

1. Clone the repository
   git clone https://github.com/yourusername/finance-tracker.git
   cd finance-tracker
2. Install dependencies
   npm install
3. Run the development server
   npm run dev

The app will start locally at:

http://localhost:5173
